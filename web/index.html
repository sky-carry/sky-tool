Features

三个高频功能，一处解决

每个功能都有全局热键，随手就能呼出，不打断你手头的事。

🔍

文件搜索

全盘文件名即时搜索，和 Everything 同原理。管理员模式下直接读取 NTFS 主文件表秒级建索引，并用 USN 日志实时跟踪增删改；普通模式自动降级为后台扫描。

Ctrl+Alt+F
✂️

截图 / 贴图

鼠标悬停自动吸附窗口边框，单击即选中。支持矩形、箭头、画笔、文字、马赛克标注，可一键复制、保存，或把截图钉在屏幕最前（支持四角拖拽缩放、双击关闭）；F3 也能把剪贴板里的图贴到桌面。

F1 截图F3 贴图

钉在桌面的今日待办，不遮挡其他窗口。每条任务可单独设提醒时间，到点弹窗提示；圆形勾选、绿色对勾、完成进度条，隔天自动清理已完成项。

Ctrl+Alt+M

Get Started

下载后怎么用

  1. 先选版本：极简版主打轻巧（搜索/截图/贴图/备忘录，无 OCR，永久免费）；Pro 版在极简版基础上多了文字提取(OCR)，后续高级功能也在 Pro。
  2. 再选安装包：「双击即用」包大但内置运行时、拿来就跑（推荐）；「精简包」体积小（极简版仅约 0.3 MB），但首次运行需先装一次 .NET 8 桌面运行时。同一版本两种包功能一致，空闲内存都很低（约 30 MB）。
  3. 双击 exe，程序会启动并最小化到系统托盘（右下角工具箱图标）。
  4. 首次运行若被 SmartScreen 拦截，点「更多信息 → 仍要运行」即可（自编译应用未做数字签名，属正常现象）。
  5. 右键托盘图标可看到全部功能和当前热键，双击图标打开主面板。
  6. 想要 Everything 那样的秒级搜索，在搜索窗口点「以管理员重启」即可启用 MFT 极速索引与实时更新。
💡 数据（备忘录、设置）保存在 %AppData%\SkyTool，卸载只需删掉 exe 即可，干净无残留。
